* 기본 코드 사용 
  > 문법에서 Contant 형은 `let age: Int = 20` 과 같이 정의를 한다. 변수는 `var number: Int = 30` 과 같이 표현하며, 데이터 타입은 변수 명 뒤에 : 과 함께 대문자로 시작하는 데이터 타입을 지정. 
```swift 
let age: Int = 20        
var number: Int = 30




```
